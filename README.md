# Forto

Forto is an all-in-one **task tracker, habit tracker, and pomodoro timer**. It keeps track of your daily tasks and habits along with your focus time, allowing everything to happen in one place.

## How to Run the Project
Follow the below steps to run Forto on your local.

### Clone the Repository
Use `git clone` to download the project:
`git clone https://github.com/your-username/your-repo-name.git`

### Go into the project folder
Use the cd command to move into the Forto project folder so you can run all project commands from the correct place.

Use the command `cd forto/forto` along with your path, for example `cd /Users/your-username/repo-folder/forto/forto`

### Install Dependencies
Use `npm install` to install all required packages:

### Run to localhost
Start the app using `npm run dev`:

Then open this in your browser (or the other localhost link shown after running the command):
`http://localhost:5173`
